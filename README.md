# FlightLog

A flight log app, built using Google's Material Design UI elements.

## Dependencies

* ruby 
	- `brew install ruby` or `yum install ruby` or `apt-get install ruby`
* ruby sass 
	- `gem install sass`
* gulp and yeoman
	- `npm install -g gulp yo`
